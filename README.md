### Setup Requirements

- node at least v10
- npm
- Your favourite browser

### Getting Started

- Clone this repo
- Install dependencies in apps(host: container, remote: auth, dashboard, marketing), `npm install`
- Run apps `npm start`
- you should be able to access the host and remote apps in your browser
